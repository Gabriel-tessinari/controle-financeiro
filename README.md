# Controle

Projeto Node.js + Express + TypeScript para gestão de caixa e estoque.

## Requisitos

- Node.js 18+
- PostgreSQL

## Instalação

```bash
npm install
```

## Rodando em desenvolvimento

```bash
npm run dev
```

## Compilando para produção

```bash
npm run build
```

## Rodando em produção

```bash
npm start
```
